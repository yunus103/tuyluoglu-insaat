## Border Radius

| Kullanım                                 | Değer          |
| ---------------------------------------- | -------------- |
| Büyük yapılar (fotoğraf, section sınırı) | `0px` — keskin |
| Kart, buton, input                       | `8px`          |
| Badge, tag                               | `4px`          |

**Genel his:** Ağırlıklı keskin, etkileşimli elemanlarda hafif yuvarlatılmış.
