# Logo Klasörü

Bu klasöre logo dosyasını koy:

- `logo.png` → Ana logo (koyu zemin OLMAYAN yerler için)
- `logo-dark.png` → Opsiyonel: koyu footer/section üzerinde kullanılacaksa

**Önerilen boyut:** 400 × 120px (yatay), şeffaf PNG veya SVG

Header öncelik sırası:
1. `/public/images/logo/logo.png` (bu dosya)
2. Sanity → siteSettings → logo
3. siteName text fallback
